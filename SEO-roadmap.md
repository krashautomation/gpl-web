# SEO Roadmap

A concise, prioritized roadmap for maximum efficiency, SEO, and traffic growth for your site and dynamic page system.

---

## Phase 1 — Immediate Wins (Weeks 1–2)

### Pillar Page Optimization

- Optimize /gold-price as the "Gold Price Live" pillar page.
- Include live chart, historical data, FAQs, and related market links.
- Target main keyword & variations.

### Internal Linking Framework

- Define rules to link clusters → pillar, clusters → clusters.
- Ensure anchor text uses natural keyword variations.

### Chart Page Protection

- Mark chart pages with isChartPage: true in CMS to avoid overwrites.

---

## Phase 2 — Rapid Content Expansion (Weeks 2–6)

### Cluster Page Creation

- Generate 10–20 AI-written cluster pages (text + optional charts/images).
- Auto-populate components, metadata, and internal links.

### Dynamic Page System

- Build JSON-driven component system (text, images, charts, tables, FAQ).
- Templates: default, chart, hybrid.

### SEO Automation

- Auto-generate meta tags, schema, canonical URLs, OG/Twitter cards for all pages.
- **Structured Data:** Implement JSON-LD schemas (Breadcrumb, Article, FAQPage, Organization, WebSite). See [`docs/structured-data-plan.md`](docs/structured-data-plan.md) for details.

---

## Phase 3 — Scale & Automation (Month 2–3)

### Automated Publishing

- Schedule 1–2 new cluster pages/day.
- Optionally, refresh old cluster pages periodically with AI.

### Content Discovery / Crawler

- Feed relevant industry/news content into AI pipeline for new cluster ideas.

### Backlink & External Signals (Optional)

- Monitor competitor backlinks, submit pillar pages to directories, outreach.

---

## Phase 4 — Advanced Scaling (Month 3+)

### Topic Pillar Network

- Add 5–10 pillars for related keywords (gold ETFs, silver, platinum).
- Repeat cluster creation, linking, and automation for each pillar.

### Analytics & Optimization Loop

- Track traffic, rankings, CTRs → optimize internal linking & content accordingly.
- Expand high-performing clusters with follow-up content.

---

## Priority Order for Maximum Impact

1. Pillar page optimization + chart page protection
2. Internal linking framework
3. Cluster page creation & dynamic page system
4. SEO automation (meta, schema, canonical)
5. Automated publishing
6. Topic pillar network & content refresh
7. Backlink/external signals & analytics optimization

---

## Key Principle

First build authority → then scale content → then automate & refresh.

This ensures Google sees topical authority early, giving your AI-driven content the fastest path to traffic.
